- git 꼭 외워야 하는 용어들
Repository : 프로젝트와 그 기록을 추적하는 폴더
Clone : 컴퓨터에 원격 저장소의 사본을 만듭니다
Stage : Git에 다음에 저장할 변경 사항을 알려줍니다
Commit : 스테이징된 변경사하의 스냅삿을 저장합니다
Branch: 동시에 여러버전이나 기능을 작업합니다
Merge: 서로다른 브런치의 변경 사항을 결합합니다
Pull: 원격저장소에서 최신 변경사항을 가져옵니다
Push: 변경사항을 원격 저장소로 보냅니다

git 명령어 
ls : 현재 폴더에 있는 파일을 나열합니다
git config --global "code --wait" vscode를 기본 편집기로

초기설정시
git config --global user.name "Your Name"
git config --global user.email "you@example.com"

깃에 모든 설정을 나열할때
git config --list

설정해제
git config --global --unset code.editor

구성수준
모든 : git config --system
글로벌 : git config --global
로컬  : git config --local

우선순위
로컬(현재저장소) git config user.name "Project Name"
글로벌(현재사용자) git config --global user.name "Global Name"
시스템(모든사용자) git config --system user.name "System Name"
